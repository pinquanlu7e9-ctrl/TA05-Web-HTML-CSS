ASIXC – Cybersecurity student

I study at Institut Tecnològic de Barcelona (ITB) in the ASIXC (Ciberseguretat) programme. I use this small website to show a few web projects related to networking and security.

Hi, I'm Pin Quan Lu

I enjoy technology, coding and learning how systems work. In ASIXC we work with networks, operating systems and basic cybersecurity tools. This portfolio is a place to practise HTML, CSS and JavaScript while connecting it to what I study.

The aim of this project is not to build a complex application, but to show that I can structure a website with semantic HTML, style it with responsive CSS and add useful interactions such as a dark mode switch, form validation and dynamic content. These are the same front-end skills that are useful when creating small dashboards or tools for cybersecurity labs.
