Project 2 – Security Notes Blog

This project is a simple multi-page blog layout. The idea is to have a clean place where I can publish short articles about what we learn in class: for example how TCP/IP works, how to harden a Linux server, or first steps with security tools.

The blog is built with semantic HTML: each post can be an <article>, grouped inside sections. CSS is used to keep the text readable and to make the layout responsive on smaller screens.

A dark mode switch is included using JavaScript and localStorage, which makes the blog more comfortable to read during late-night study sessions.
