Project 1 – Cyber Lab Task Board

This project is a small web application that works as a “task board” for cybersecurity labs. Each card on the board represents a lab task: for example capturing packets with Wireshark, configuring firewall rules, or scanning a network segment.

The goal is not to implement a complex backend, but to practise HTML structure, CSS layout and basic JavaScript interactions. Tasks can be marked as “To do”, “In progress” or “Done” by moving them between columns.

This is useful during ASIXC practical sessions, because it helps to keep track of which activities are finished and which still need work.
