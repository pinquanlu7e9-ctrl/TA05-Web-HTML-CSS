项目一：在线书店网站

该项目是一个模拟的在线书店网站，提供浏览和购买图书的功能。用户可以浏览书籍列表，查看每本书的详细信息，并将想购买的书籍加入购物车。

在这个项目中，我主要负责前端页面的开发和设计。通过使用 HTML 和 CSS 实现了整洁的页面布局，使用 JavaScript 实现了购物车交互、筛选搜索等功能。此外，页面采用了响应式设计，以兼容不同设备的屏幕尺寸。

这个在线书店项目展示了一个典型的电商网站前端结构。我在项目中学习并应用了：

  • HTML5 语义化标签组织页面结构
  • CSS3 布局和样式（包括 Flexbox 布局和媒体查询）
  • JavaScript 实现交互效果（如购物车动态更新）

通过该项目，我对 Web 应用的前端开发有了更深入的理解，也锻炼了独立开发和调试的能力。

项目摘要

开发角色：前端开发

使用技术：HTML, CSS, JavaScript
